  Class types
  ------------
  In java representing primitie types we have equivalent class types
  those are called wrapper classes
  
  byte - Byte
  short - Short
  int - Integer
  long - Long
  float - Float
  double - Double
  char - Character
  boolean - Boolean
  String - String
  
  for wrapper classes the default value is null
  
  int i; //default value is 0
  Integer i; //default value is null
